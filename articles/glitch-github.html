Glitch and Github Pages

So I wanted to start a simple website with articles and a wiki. Glitch is great but I like sharing code on Github. I also might want to change my setup and Github pages are easy to publish to from lots of software.

I decided to use Github pages as the hosting for my website but wanted to connect it to Glitch to use it's development environment.

Creating the project

  1. Create a repository in Github and use GitHub Pages. [See setup instructions]
  2. Start a new Glitch Project by pulling from a Github repository.

Now you should have a project to start playing with.
